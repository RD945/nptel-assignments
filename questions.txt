QUESTION 1: [1 mark]
ELMo and BERT represent two different pre-training strategies for language models. Which
of the following statement(s) about these approaches is/are true?
a) ELMo uses a bi-directional LSTM to pre-train word representations, while BERT uses
a transformer encoder with masked language modeling.
b) ELMo provides context-independent word representations, whereas BERT provides
context-dependent representations.
c) Pre-training of both ELMo and BERT involve next token prediction.
d) Both ELMo and BERT produce contextual embeddings usable for downstream tasks.
Correct Answer: a, d
Solution:
(a) True. ELMo’s base model is a bidirectional language model implemented with stacked
LSTMs (forward and backward LMs). BERT is a Transformer encoder trained with Masked
Language Modeling (MLM) and Next Sentence Prediction (NSP).
(b) False. ELMo produces context-dependent token representations (its pooled
representations depend on surrounding tokens via the BiLM). BERT also produces contextdependent representations. The statement claims ELMo is context-independent - that is
incorrect.
(c) False. ELMo is trained with language-modeling objectives (predict next token in forward
LM and previous token in backward LM), so it uses next-token prediction in its LM training.
BERT, however, uses masked LM (MLM) rather than standard next-token prediction; it
predicts a masked token given context. Thus “both involve next token prediction” is incorrect.
(d) True. Both produce contextual embeddings that downstream tasks can use and adapt:
ELMo provides layerwise LM activations that tasks combine; BERT is fine-tuned, either endto-end or an additional classification head is learnt, for tasks.

QUESTION 2: [1 mark]
Which model family is natively designed for conditional sequence-to-sequence tasks of the
form P(y|x) without architectural modification?
a) Decoder-only language models
b) Encoder-only models
c) Encoder-decoder models
d) Character-CNN models
Correct Answer: c
Solution:

prompts; encoder-only models (like BERT) do not generate sequences autoregressively by
design.
